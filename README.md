The released code constitutes the core reconstruction framework of our work on implementing computational recovery for mid-infrared snapshot spectral imaging based on intrinsic nonlinear radial dispersion. In this system, the acquired two-dimensional snapshot measurement encodes the three-dimensional spectral datacube through wavelength-dependent radial scaling and speckle-induced spatial modulation, and the code is used to invert this highly multiplexed measurement to recover the full hyperspectral information.
Specifically, the code models the forward imaging process and performs reconstruction using a generalized alternating projection algorithm with total variation regularization (GAP-TV), enabling efficient recovery of spectral and spatial features from compressed measurements. Our implementation is adapted and further developed from an existing open-source CASSI reconstruction framework, with modifications tailored to the nonlinear radial dispersion mechanism and the corresponding sensing model.
We gratefully acknowledge Prof. Xin Yuan (Westlake University) for generously sharing the original codebase, which served as an important foundation for this work, as well as the open-source resources that facilitated the development of our reconstruction pipeline. The code can be accessed at: https://github.com/zsm1211/PnP-CASSI.
